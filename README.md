## "뷰잉은 면접스터디를 위한 통합 솔루션입니다."

### 👨‍👩‍👧‍👦 사용자

면접 스터디 구인에 어려움을 겪는 사람<br>
면접 관련 커뮤니티가 필요한 사람<br>
면접 연습을 하며 피드백을 주고받고 싶은 사람

### 💁 사용방법

면접을 압둔 기업의 이름을 검색해보세요.<br>
원하는 스터디가 있다면 가입 신청을 해보세요.<br>
없다면 직접 스터디 생성하기 !
